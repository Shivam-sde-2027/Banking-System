Banking Management System (C++ | OOP)

A console-based Banking Management System developed using C++ and Object-Oriented Programming (OOP) principles.
The project simulates real-world banking operations such as account creation, deposits, withdrawals and interest calculation with proper exception handling and modular design.

Project Overview

It copys essential banking functionalities while maintaining clean architecture, data encapsulation, and error handling.

Key Features
-Account Management
 -Create Savings & Current accounts
 -Unique account number generation
 -Customer details storage
-Core Banking Operations
 -Deposit money
 -Withdraw money (with balance validation)
 -Check account balance
 -Account summary display
-Interest Calculation 
 -Savings account interest support
 -Polymorphic interest computation
-Exception Handling
 -Insufficient balance handling
 -Invalid account number detection
 -Invalid input validation
 -Runtime safety using try-catch

-OOP Concepts Implemented

 -Encapsulation – private data with public interfaces
 -Abstraction – base Account class
 -Inheritance – SavingsAccount, CurrentAccount
 -Polymorphism – virtual functions for interest calculation
 -Static members – account number generation
 -STL usage – vector, map
 -File handling (optional) – persistence support
 -Menu-Driven Interface
